# 🌊 SamudhraVani – Marine Conservation & Ocean Awareness Platform

## 📌 Overview

SamudhraVani is a web platform created to promote awareness about marine biodiversity, ocean conservation, and sustainable living. The project combines educational content, interactive learning experiences, and real-time information to engage users in protecting life below water.

This was my **second major web development project**, built after learning the fundamentals of HTML, CSS, JavaScript, and backend development with Node.js. The goal was to move beyond static websites and experiment with dynamic content, API integration, and user interaction.

---

## 🎯 Project Goals

* Raise awareness about marine conservation
* Promote SDG 14: Life Below Water
* Provide interactive learning through games and quizzes
* Display ocean-related news from external sources
* Allow users to participate through volunteer registration
* Explore full-stack web development concepts

---

## 🚀 Running the Project

### Clone Repository

```bash
git clone https://github.com/yourusername/samudhravani.git
cd samudhravani
```

### Install Dependencies

```bash
npm install
```

### Start Server

```bash
npm start
```

### Open Browser

```text
http://localhost:3000
```
